## Featured Project

### AcceleronOS (Bare-Metal 16-bit OS)
This is my core project—a completely custom, monolithic kernel Operating System built entirely in **NASM x86 Assembly**. It runs in 16-bit Real Mode on a virtual floppy disk.

* **Status:** Currently developing v1.0.5 (Implementing FAT16 Filesystem support).
* **Key Achievement:** Built a custom bootloader, kernel, and a functional Command-Line Interface (CLI) with command parsing (`clr`, `ver`, `help`).
* **Technical Stack:** NASM Assembly, BIOS Interrupts (`INT 10h`, `INT 13h`, `INT 16h`), QEMU.
* **Repository:** github.com/aritrash/AcceleronOS
